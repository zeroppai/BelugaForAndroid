BelugaForAndroid
================

Android対応のBelugaクライアントです。

- 投稿/閲覧
- クイックおっぱい投稿
- ルームのタブ管理
- 画像アップロード機能

###動作確認バージョン
Android 2.2, 2.3, 4.0, 4.1、で動作確認されています。
なにかあれば[@0](http://beluga.fm/user/0)にリプライください。
